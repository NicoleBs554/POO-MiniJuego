# Study Break Runner

Un juego de carrera infinito con temática académica donde escapas de profesores y recolectas asignaciones.

## Características

- Jugador con movimientos fluidos (correr, saltar)
- Generación procedural de obstáculos y enemigos
- Sistema de colisiones
- Múltiples pantallas (Menú, Juego, Pausa, Game Over)
- Sistema de puntuación
- Gráficos simples pero efectivos

## Controles

- **ESPACIO**: Saltar
- **FLECHA IZQUIERDA/DERECHA**: Moverse
- **P**: Pausar/Reanudar
- **ESC**: Salir al menú
- **R**: Reiniciar (en pantalla de Game Over)

## Requisitos

- Java 11 o superior
- Maven (para compilar)
